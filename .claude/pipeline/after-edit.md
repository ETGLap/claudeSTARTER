# Pipeline: after-edit

Layers 7–12. After code is written.

- Run the tests; make them pass (Green). Read the output.
- Refactor the code you just wrote while green (`reviewers/refactoring.md`, in-pipeline mode).
- Review quality (`reviewers/quality.md`) and security (`reviewers/security.md`).
- Check placement and consistency (`reviewers/architecture.md`).
- Re-run tests after refactor; confirm no regressions.
- Document what was verified and what was not.
